## 3. Процесс очистки данных

### 3.1 Конвертация валют
- **Загружены курсы** с `https://open.er-api.com/v6/latest/USD`
- **Использованные курсы**:
  - 1 USD = 490.00 KZT (прибл., резервное значение при недоступности API)
  - 1 USD = 83.00 INR (прибл., резервное значение при недоступности API)
  - INR → KZT: умножить на (KZT/USD ÷ INR/USD)

### 3.2 Валидация данных
**Удалены строки с**:
- Отсутствующими названиями товаров или длиной < 3 символов
- Отсутствующими или неположительными ценами
- Некорректными ценовыми данными (нечисловые)

### 3.3 Извлечение бренда
- Применены **50+ regex-паттернов** для извлечения брендов из названий товаров
- Паттерны охватывают крупнейших производителей электроники:
  - Apple, Samsung, Sony, LG, Google, OnePlus, Xiaomi и т.д.
  - Бренды, не распознанные при извлечении, помечены пустой строкой
  - Резервный вариант: использован предоставляемый API бренд в случае отсутствия

### 3.4 Обнаружение дубликатов
**Логика ключа дедупликации**:
- Если URL товара существует: использовать URL (нормализованный, нижний регистр)
- Иначе: использовать хеш (название_товара + бренд + цена_kzt)

**Результат**: 0 дубликатов удалено (источники данных уже были чистыми)

### 3.5 Обнаружение выбросов (метод 1.5xIQR)
**Процесс**:
1. Вычислить Q1 (25-й перцентиль) и Q3 (75-й перцентиль) для `price_kzt`
2. IQR = Q3 - Q1
3. Нижний предел = max(Q1 - 1.5×IQR, 0)
4. Верхний предел = Q3 + 1.5×IQR
5. Отметить строки, где цена < нижний_предел ИЛИ цена > верхний_предел

**Результаты**:
- Диапазон IQR: **0 – 19,555 KZT**
- Отмечено выбросов: **717 строк** (14.3%)
- Действие: отмечены, но **НЕ удалены** (экстремальные цены — реальные премиум-товары)

### 3.6 Сегментация по цене
Товары разделены на 4 категории по цене в KZT:

| Сегмент | Диапазон цены | Количество | % |
|---------|---------------|-----------|-----|
| низкобюджетный | < 80,000 KZT | 4,731 | 94.3% |
| среднесегмент | 80,000 – 250,000 | 238 | 4.7% |
| премиум | 250,000 – 600,000 | 38 | 0.8% |
| люкс | ≥ 600,000 | 10 | 0.2% |

## 5. Реализация кода

### Основной скрипт pipeline: `pipeline.py`, `pipeline2.py`)

**Ключевые функции**:
- `fetch_fx_kzt_inr_per_usd()` — загрузка курсов валют (с резервным вариантом)
- `load_github_electronics()` — парсинг CSV Amazon India
- `load_dummyjson()` — запрос к DummyJSON API
- `load_modern_gadgets_seed()` — загрузка seed-данных
- `merge_and_clean()` — объединение и очистка всех источников
- `extract_brand()` — извлечение бренда через regex

**Запуск pipeline**:
```bash
python pipeline.py
# или быстрый тест:
python pipeline.py --quick
```

**Выходные файлы**:
- `qazaqprice_dataset.csv` — очищенный датасет
- `data_cleaning_log.txt` — подробный лог всех шагов

### Анализ данных: `analysis.py`

Создаёт визуализации:
- распределение цен (гистограмма)
- разбор по ценовым сегментам (бар-чарт)
- топ брендов (горизонтальный бар)
- топ категорий (бар-чарт)
- доля выбросов (круговая диаграмма)

**Запуск анализа**:
```bash
python analysis.py
```

**Папка вывода**: `plots/` с PNG-графиками и текстовым файлом резюме
